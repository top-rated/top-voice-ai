LinkedIn GPT API Documentation

This document provides detailed information about the LinkedIn GPT API endpoints with ready-to-use curl commands for testing.

Base URL: https://gpt.top-rated.pro/api/v1

Authentication

Register a new user

Register a new user and get a subscription ID (free tier).

POST /auth/register

curl -X POST https://gpt.top-rated.pro/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "name": "John Doe"
  }'

Example response:

{
  "message": "User registered successfully",
  "subscriptionId": "free_1234567890_abcdefg",
  "subscriptionType": "free",
  "user": {
    "email": "user@example.com",
    "name": "John Doe"
  }
}

Login user

Login with email to retrieve a subscription ID.

POST /auth/login

curl -X POST https://gpt.top-rated.pro/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com"
  }'

Example response:

{
  "message": "Login successful",
  "subscriptionId": "free_1234567890_abcdefg",
  "subscriptionType": "free",
  "user": {
    "email": "user@example.com",
    "name": "John Doe"

}

Verify subscription

Verify if a subscription ID is valid.

GET /auth/verify-subscription

curl -X GET "https://gpt.top-rated.pro/api/v1/auth/verify-subscription?subscriptionId=free_1234567890_abcdefg"

Example response:

{
  "active": true,
  "type": "free",
  "features": {
    "canSearch": false,
    "canAnalyzeProfiles": false,
    "searchLimit": 0,
    "profileLimit": 0
  }
}

Profile Analysis Premium

Analyze LinkedIn profiles

Analyze one or more LinkedIn profiles.

POST /profiles/analyze Premium

curl -X POST https://gpt.top-rated.pro/api/v1/profiles/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "profileUrls": [
      "https://www.linkedin.com/in/janedoe/",
      "https://www.linkedin.com/in/johndoe/"
    ],
    "subscriptionId": "premium_1234567890_abcdefg"
  }'

Example response:

{
  "message": "Profile analysis started",
  "requestId": "req_1234567890_abcdefg",
  "status": "pending",
  "estimatedTime": "10 seconds"
}

Search Premium

Direct search (GET method)

Search LinkedIn posts by keywords using a GET request.

GET /search Premium

curl -X GET "https://gpt.top-rated.pro/api/v1/search?query=artificial%20intelligence&timeframe=past-week&limit=5&subscriptionId=premium_1234567890_abcdefg"

Example response:

{
  "message": "Search started",
  "searchId": "search_1234567890_abcdefg",
  "status": "pending",
  "estimatedTime": "10 seconds"
}

Search by keywords (POST method)

Search LinkedIn posts by keywords using a POST request.

POST /search/keywords Premium

curl -X POST https://gpt.top-rated.pro/api/v1/search/keywords \
  -H "Content-Type: application/json" \
  -d '{
    "keywords": "artificial intelligence",
    "timeframe": "past-week",
    "subscriptionId": "premium_1234567890_abcdefg"
  }'

Example response:

{
  "message": "Search started",
  "searchId": "search_1234567890_abcdefg",
  "status": "pending",
  "estimatedTime": "10 seconds"
}

Error Responses

The API returns standard HTTP status codes to indicate success or failure:

  • 200/201/202: Success
  • 400: Bad request (invalid input)
  • 403: Forbidden (subscription required or invalid)
  • 404: Not found
  • 500: Server error

Example error response:

{
  "message": "This feature requires a subscription",
  "subscriptionUrl": "https://linkedingpt.gumroad.com/l/subscribe?wanted=true"
}
